Expand description

Trait for safe wrappers over JS event types and JS event listening.

It is up to the implementor to ensure that the event_type is correct and associated type Data matches .name().

Required Associated Types

source

Provided Methods

source

fn add_listener(&self, target: &EventTarget) -> Listener<Self::Data>

Adds event listeners to JS’s event targets, where events are of this event type. This function is asynchronous and a future is returned.

It is up to the caller to ensure the target supports this event type.
